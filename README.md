## 📊 Overview
MetaForge TFT is an analytics platform designed to provide detailed insights into Teamfight Tactics gameplay through statistical analysis and interactive tools. The application enables data-driven decision making for strategic development and optimization.

## 📌 Key Features
- **Team Builder**: Interactive champion positioning with drag-and-drop functionality
- **Real-time Synergy Calculator**: Instant trait activation feedback
- **Item Optimization**: Smart recommendations based on performance data
- **Composition Library**: Save and share your winning strategies
- **Champion Analytics**: Performance metrics across different team contexts
- **Meta Tracking**: Track emerging compositions and win rates
- **Counter Intelligence**: Identify optimal counter-strategies
- **Patch Impact Analysis**: Understand how changes affect the meta

## ⚙️ Tech Stack
- **Frontend**: React 18, Next.js 15, TypeScript, TailwindCSS
- **State Management**: React Query, Context API
- **Data Visualization**: Recharts, D3.js
- **Interaction**: React DnD, Framer Motion, DnD Kit
- **Styling**: TailwindCSS, GSAP
- **Database**: PostgreSQL
- **Deployment**: Vercel, AWS

## 📂 Project Structure
- `/src`: Source code
  - `/components`: UI components
    - `/ui`: Base UI elements
    - `/common`: Shared components
    - `/entity`: TFT entity components
    - `/team-builder`: Team builder features
    - `/analytics`: Data visualization components
  - `/pages`: Application routes
    - `/api`: Backend endpoints
    - `/entity`: Entity detail pages
    - `/team-builder`: Team composition tools
    - `/meta-report`: Meta analysis pages
  - `/hooks`: Custom React hooks
  - `/utils`: Utility functions
    - `/db`: Database helpers
    - `/api`: API integration
    - `/auth`: Authentication utilities
  - `/mapping`: Game data mapping
  - `/types`: TypeScript definitions

## 🔗 Resource Integration
MetaForge incorporates assets from the [TFT-Assets](https://github.com/gaba-dev-1/tft-assets) repository, providing standardized game resources and data structures for the application's visualization and analysis systems.
